The reports of all the executed simulations are inside the folder Private_Utilities/reports. 
Inside syn/bin you can find the syn_nangate.tcl file needed to simulate the modified version of the circuit. 
Also inside Private_Utilities you can find script.sh to run all the commands needed to obtain the results.
In the rtl folder you can find all the files we added to hardenize the circuit. 
In sbst/tests you can find the file test1.S that is the sbst file we created to increase the coverage of our components.
